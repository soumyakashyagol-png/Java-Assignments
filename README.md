# Student Skill Tracking System

## 📌 Description
This is a Java-based project using MySQL and JDBC to manage student skills and practice activities.

## 🚀 Features
- Add Student
- Add Skills
- Track Practice Hours
- View DSA Students
- Performance Analytics

## 🛠 Tech Used
- Java
- MySQL
- JDBC

## ▶️ How to Run
1. Compile the project
2. Run App.java
3. Use menu options

## 📊 Output
Menu-driven console application
